SAPPHIRE - Sequence Analyser for the Prediction of Prokaryotic Homology Inferred Regulatory Elements

Promoter prediction software for Pseudomonas

___________________

Contact: rob.lavigne@kuleuven.be

___________________

Required python3 packages: 

- numpy, keras, biopython

Usage:

- cd to SAPPHIRE directory
- Run:
  python3 SAPPHIRE.py input.fasta
- Or (to include the reverse complements):
  python3 SAPPHIRE.py -r input.fasta
